# Growfinix Internship Task 3

## Semantic Sentiment & Urgency Analyzer

### Objective
Analyze customer feedback and identify:
- Sentiment
- Urgency
- Main Issue

### Technologies Used
- Python
- OpenAI API

### How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Replace `YOUR_OPENAI_API_KEY` with your API key.

3. Run:
   python main.py
